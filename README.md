🚀 OrbitLang

OrbitLang is a small custom programming language implemented from scratch in Python.

It is designed as a learning project to understand how programming languages work internally, including lexing, parsing, and execution.

✨ Features
Variable assignment
Print statements
If / else control flow
Basic expression handling
AST-based interpreter architecture
📌 Example
set x = 10

if x == 10 {
print 99
} else {
print 1
}
⚙️ How it works

OrbitLang works in three main stages:

Lexer – Converts source code into tokens
Parser – Builds an Abstract Syntax Tree (AST)
Interpreter – Executes the AST
🚀 Run
orbit run examples/test.orbit
📁 Project Status

This is an early working version (MVP).
The core language is stable and ready for further development.

Planned improvements include functions, scopes, and better expression parsing.
